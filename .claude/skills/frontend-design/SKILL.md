---
name: frontend-design
description: Create distinctive, production-grade frontend interfaces with high design quality. Use when building web components, pages, artifacts, dashboards, or any UI. Avoids generic AI aesthetics.
---

Before coding, commit to a BOLD aesthetic direction (brutalist, maximalist, retro-futuristic, organic, luxury, editorial, etc.). Implement working code (HTML/CSS/JS or React) that is:
- Visually striking and memorable
- Production-grade and functional
- Cohesive with a clear aesthetic point-of-view

Typography: Use distinctive, characterful fonts. NEVER Arial, Inter, Roboto, Space Grotesk.
Color: Cohesive palette with CSS variables. Dominant colors + sharp accents.
Motion: CSS animations, staggered reveals, surprising hover states.
Layout: Asymmetry, overlap, diagonal flow, grid-breaking elements.
Backgrounds: Gradient meshes, noise textures, dramatic shadows, grain overlays.

NEVER use purple gradients on white, generic fonts, or cookie-cutter layouts.
